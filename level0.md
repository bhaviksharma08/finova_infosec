# Level 0 - Bandit
## *Key Learnings:*
  - What's SSH: A network protocol to remotely login to servers and transfer files over unsecured networks
  - How to use SSH to connect to a server remotely as a different user/host:
    ##### Linux command:    
    ```ssh -p <port> <host_name>@<domain_name_of_the_website>```

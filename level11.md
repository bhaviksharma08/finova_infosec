# Bandit 10 -> 11

## Challenge: Decoding base64 data in a file

### Command:
```bash
base64 -d data.txt
```

### Explanation:
- ```base64``` utility in linux: lets you work with base64 data
- ```-d``` argument: takes a base64 file and decodes it
